## My first app 🚀

This is officially my first app created using react native. Context: the app is simple, the main function is access the database and show the user latest data received from microcontroller esp32. The data informed is about values from two different sensors. By doing that, I added some basics as step to user login, so we have the first step: user verification, and the manipulation to database.
